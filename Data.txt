## 📝 ExamContext.jsx

**State:** `exam` (full exam+questions), `loading`, `error`, `currentQuestionId`, `answers{}`, `visitedQuestions` (Set)

**localStorage keys** (per examId+uid):
- `exam_{id}_{uid}_answers` · `_position` · `_timeLeft`

**Logic on load:**
1. Fetches full exam via `getFullExam`
2. Runs `checkExamAccess` (active, time window, grade)
3. Checks `checkAlreadySubmitted` → redirects to `/results/{examId}` if true
4. Restores answers + position from localStorage if available

**Functions:**
- `markVisited(questionId)` — adds to visited Set
- `clearExamStorage()` — removes all 3 localStorage keys for this exam

**Hook:** `useExam()` → exposes `{ exam, loading, error, currentQuestionId, setCurrentQuestionId, answers, setAnswers, visitedQuestions, markVisited, clearExamStorage }`
